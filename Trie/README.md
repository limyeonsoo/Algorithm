# Trie

---

문자열 1개 씩 node에 저장하여 탐색 할 수 있는 방법.

시간복잡도는 문자열의 길이 만큼이지만

공간복잡도는 복잡하다.
  숫자 :  1노드당 10개의 0~9 노드를 가짐.
  영어 :  1노드당 26개의 a_z 노드를 가짐.
